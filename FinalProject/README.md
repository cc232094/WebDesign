While building my site, I learned how much of a difference small changes like animations and transitions can make to the overall cohesiveness and style of a site. I balanced creativity with usability by keeping my palette consistent and uniform, both to express a desired style and to improve readability, as well as utilizing various different images, allowing for visual expression while also making it easier to distinguish pages from one another. I am most proud of my navigation bar because it was one of the more difficult elements to change across pages and I was very happy to get the centered text, dropdown, and help button all working despite difficulties. I see myself using these skills later on when considering the visual design process, mainly for interfaces and menus, when developing games.
